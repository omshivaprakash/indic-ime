== Description ==

Using IndicIME plugin, you can very easily write in 9 Indian scripts. It does not require any software download for writing in Indian script. Using this plugin, visitor can directly type in Indic script in search box or comment. Blog owner now can directly type title of the post in his langauge of choice. If you need any helping in typing, a quick reference image is shown when clicked question mark icon. 

Typing is extremely easy. Typing follows "The way you speak, the way you type" rule. It is called transliteration. For example, choose "Devnagari" from the script list and try to if you type "bhaarat". You will see that depending on the key you pressed, it will automatically be converted into devnagari script. Half letters and special letters are also very easy to write. Try to type "prashna", "karma" or "padma" and you will see how easy it is to type in your favourite language. For reference of English and Indian script character mapping, please refer to the help by clicking on help (/question mark) icon. 

== Installation ==

1. Unzip the file on your local hard disk.
1. Upload "indicime" folder to BLOGROOT/wp-includes/js/tinymce/plugins/
1. Upload the file "indicime.php" to BLOGROOT/wp-content/plugins directory.
1. Activate via WordPress Plugins tab.

Make sure that in your theme, in footer.php file wp_footer() function call exists.

== Frequently Asked Questions ==

= This plugin is not working at all =
Make sure that in your therem in footer.php file wp_footer() function call exists.
= I cannot see IndicIME bar on upper right corner =
You may not have any textbox to type in. IndicIME plugin is smart enough to hide itself when not needed.
=Everything is correct but still I do not see IndicIME toolbar =
Your Web browser might have cached all your javascript. Make sure to purge cache by refreshing it. You may press ctrl+F5 to reload the page.
